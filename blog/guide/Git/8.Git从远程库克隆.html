# Git 从远程库克隆

如果已经有了远程仓库，或者想要从GitHub上面拉取别人的仓库，也就是从远程克隆。

首先，可以删除之前我们创建再本地的GitDemo文件夹，然后我们登录GitHub，搜索我们之前创建的 “GitDemo”，进入页面，像下面一样操作，复制地址。

回到桌面，右键打开 “Git Bash” 输入以下内容：

git clone https://github.com/lfb0728/GitDemo.git // 克隆远程仓库

这时桌面就有一个和远程一样的版本库了。
